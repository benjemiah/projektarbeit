## Projektbeschreibung


Dieses Projekt stellt eine einfache HTML-Seite mit einem Login-Formular dar. Der Benutzer kann seine Anmeldedaten eingeben (Loginname und Passwort). Bei korrektem Login werden sie auf eine andere Seite weitergeleitet.


## Hauptfunktionen

- Benutzer können ihre Anmeldedaten eingeben.

- Ein "Test-Login" ist implementiert (Benutzername: testuser, Passwort: testpass).

- Nach erfolgreichem Login erfolgt eine Weiterleitung zu einer neuen Seite (seite.html), nach einer kurzen Verzögerung von 5 Sekunden.

## Zielgruppe

Diese Demo ist für Entwickler gedacht, die ein einfaches Login-Formular implementieren möchten.

## Verwendete Sprache/Skripte

HTML

- HTML: Die Struktur der Seite, einschließlich des Login-Formulars.

- CSS: Grundlegendes Styling der Login-Seite.

- JavaScript: Überprüft die Eingabedaten und steuert die Weiterleitung zur nächsten Seite.

## Projektstruktur

├── index.html        # Die Hauptdatei mit der Weiterleitung zum Login Formular
└── andereSeite.html  # Login Formular
└── seite.html        # Seite nach erfolgreichem Login

## Dateien im Detail

- index.html

Enthält eine Website gefüllt mit Lorem-Ipsum Text sowie einen Button über den man Zugriff erhält auf das Login Formular

- andereSeite.html

Enthält das Login Formular, bei dem sich der Benutzer mit einem Passwort und einem Benutzernamen anmelden kann

- seite.html

Enthält Text als visuelle Bestätigung eines erfolgreichen Logins!

## Installation/Ausführung
